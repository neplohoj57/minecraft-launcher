# 🔑 Как получить API ключ для облачной синхронизации

## 1. Регистрация на JSONBin.io

1. Перейди на сайт: https://jsonbin.io
2. Нажми **"Sign Up"** (Регистрация)
3. Введи email и пароль
4. Подтверди email

## 2. Получение API ключа

1. После входа нажми на **свой профиль** (правый верхний угол)
2. Выбери **"API Keys"**
3. Нажми **"Create Access Key"**
4. Введи название: `Minecraft Launcher`
5. **Скопируй ключ!** (он начинается с `$2a$10$...`)

## 3. Использование в лаунчере

1. Запусти лаунчер
2. Нажми кнопку **"☁️ Облачная синхронизация"**
3. Вставь API ключ в поле
4. Нажми **"Сохранить ключ"**
5. Готово! 🎉

## 4. Первая синхронизация

1. Нажми **"⬆️ Загрузить в облако"** - сохранит текущие настройки
2. Включи **"🔄 Автосинхронизация"** - будет синхронизировать каждые 5 минут

## 5. Использование на другом ПК

1. Установи лаунчер
2. Войди в тот же аккаунт
3. Открой облачные настройки
4. Введи тот же API ключ
5. Нажми **"⬇️ Скачать из облака"**
6. Все настройки загрузятся! 🚀
