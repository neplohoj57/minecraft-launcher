# 🎮 Minecraft Launcher

Современный лаунчер Minecraft с системой аккаунтов, автообновлениями и кастомными скинами.

## ✨ Возможности

- 🔐 **Система аккаунтов** - регистрация и вход с хешированием паролей (SHA256)
- 🎨 **Кастомные скины** - загрузка своих скинов (PNG 64x64 или 64x32)
- 🔄 **Автообновления** - автоматическая проверка и установка обновлений через GitHub Releases
- ⚙️ **Настройки запуска** - выбор версии Minecraft (1.8.9 - 1.20.4) и выделение RAM (2-8 GB)
- 💾 **Сохранение сессии** - автоматический вход после авторизации
- 🎯 **Простой интерфейс** - современный дизайн с градиентами и анимациями

## 📋 Требования

- **ОС:** Windows 10/11 (x64)
- **Node.js:** 18.0.0 или выше
- **RAM:** минимум 4 GB (рекомендуется 8 GB)
- **Место на диске:** минимум 2 GB свободного места

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/neplohoj57/minecraft-launcher.git
cd minecraft-launcher
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск лаунчера

```bash
npm start
```

### 4. Сборка установщика (опционально)

```bash
npm run build
```

После сборки установщик появится в папке `dist/`.

## 🎮 Как начать играть

1. **Зарегистрируйтесь** или войдите в существующий аккаунт
2. **Выберите версию** Minecraft в левом меню
3. **Настройте RAM** (рекомендуется 4 GB)
4. **Нажмите "ИГРАТЬ"** и дождитесь загрузки файлов
5. **Наслаждайтесь игрой!**

## 👤 Управление профилем

- Нажмите на аватар или имя пользователя для открытия профиля
- Выберите один из предустановленных аватаров
- Загрузите свой скин (PNG, 64x64 или 64x32 пикселя)
- Выйдите из аккаунта при необходимости

## ⚠️ Важные замечания

- Лаунчер использует оффлайн-режим (не требует лицензию Minecraft)
- Все данные хранятся локально в `%APPDATA%/minecraft-launcher`
- Пароли хешируются алгоритмом SHA256 перед сохранением
- Для корректной работы требуется стабильное интернет-соединение
