# 🎮 Minecraft Launcher

Простой и красивый лаунчер для Minecraft на Electron

## ✨ Возможности

- ✅ Красивый GUI с фоном Minecraft
- ✅ Сохранение настроек (никнейм, версия, память)
- ✅ Выбор версии Minecraft
- ✅ Настройка оперативной памяти
- ✅ Прогресс бар загрузки
- ✅ Офлайн режим

## 🚀 Установка для разработки

```bash
npm install
npm start
